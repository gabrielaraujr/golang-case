Você é um **Engenheiro de Sistemas** e foi designado para atender um grande projeto de Transformação Digital, onde o objetivo é construir um app para captura e análise de propostas de novos clientes a plataforma de Cartões.

Considere no escopo:

- Cadastro de clientes/conta/cartão
- Análise de Documentos
- Análise de Crédito
- Análise de Fraude
- Notificação do cliente dos passos da proposta

Imagine que você irá apresentar sua proposta em uma banca de projetos e será avaliada por Arquitetura, Engenharia e Desenvolvimento.

**Stack utilizadas**:
- Golang 1.25
- PostgreSQL 18.1
- Docker & Docker Compose
- AWS SQS (LocalStack para desenvolvimento local)
- Chi Router (HTTP)

**Arquitetura proposta**:
- Microserviços
- Hexagonal
- Clean
- Cloud
- 12 Fatores
